# Instruções para deploy no Heroku

Para solucionar o problema de H10 no Heroku, siga os passos abaixo:

1. Execute os seguintes comandos no terminal:

```
heroku buildpacks:clear
heroku buildpacks:set heroku/nodejs
```

2. Configure as seguintes variáveis de ambiente:

```
heroku config:set NODE_ENV=production
heroku config:set NPM_CONFIG_PRODUCTION=true
heroku config:set FOR4PAYMENTS_SECRET_KEY=seu-token-aqui
```

3. Certifique-se de que o arquivo `Procfile` contém apenas:

```
web: npm run start
```

4. Faça deploy com o seguinte comando:

```
git push heroku main
```

5. Verifique os logs com:

```
heroku logs --tail
```

Caso continue tendo problemas, tente reiniciar a aplicação:

```
heroku restart
```

Você também pode verificar o status dos dynos com:

```
heroku ps
```

# Nota importante:
- Não tente usar o buildpack Python e Node.js juntos para simplificar
- Estamos usando apenas o serviço Node.js que está contido no Procfile
